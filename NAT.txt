DYNAMIC NAT(Many to one):
1. Interfaces status
interface fastEthernet 0/0	[public ip address -> outside]
ip nat outside
exit

int fastEthernet 0/1		[private ip address -> inside]
ip nat inside 
exit

2. Excess control list -> private ip match public ip
access-list 1 permit 192.168.1.0 0.0.0.255	[wildcard mask]

3. public ip's pool
ip nat pool RR ?
	A.B.C.D Start IP Address
ip nat pool RR 200.1.1.200 ?
	A.B.C.D END IP Address
ip nat pool RR 200.1.1.200 200.1.1.200 ?
	netmask Specify the network mask
ip nat pool RR 200.1.1.200 200.1.1.200 netmask 255.255.255.0

ip nat inside source list 1 pool RR overload	[ip traffic-> translate(pool RR-> overload)]
